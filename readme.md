# PyForce: Brute force password cracker for Excel files

## Setup
```powershell
cd C:\source
git clone https://github.com/Simon-Charest/pyforce.git
cd pyforce
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Run
```powershell
cd C:\source\pyforce
.\.venv\Scripts\Activate.ps1
python .
```
